HelloApp is a Java console application that demonstrates basic programming concepts step by step.

Features:
- Hello World output
- Command-line input
- Multiple command-line names
- Keyboard input
- Multiple keyboard names
- Store names in ArrayList
- Remove names from list
- Refactor program into methods
- Separate NameManager class
- Save names to file
- Banner style greeting output

How to run:

Compile:
javac HelloApp.java NameManager.java

Run:
java HelloApp
